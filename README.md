
# Flutter Login OTP

A simple mobile app for user authentication and OTP validation.




## Tech Stack
- **Database** : SqfLite (SQL Local Database)
- **State Management** : Riverpod
- **Router** : Go Router
- **Session** : Shared Preferences (to save user login)
